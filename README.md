# 🏢 University HR Management System

A comprehensive, full-stack Human Resources Management System designed to streamline academic and administrative tracking. Built with classic ASP.NET, C#, and a highly optimized SQL relational database.

## ✨ Key Features
* **Relational Database Architecture:** Strictly normalized (3NF/BCNF) database schema to ensure complete data integrity.
* **Performance Optimization:** Utilizes complex SQL queries, views, and Stored Procedures to handle heavy data mutations and enforce business logic at the database level.
* **Attendance Tracking:** Automated modules for logging and tracking university employee attendance.
* **Dynamic Payroll Calculations:** Secure processing and calculation of employee payroll based on attendance and system records.
* **Administrative Dashboard:** Secure, interactive web endpoints built with ASP.NET for managing leaves of absence and personal HR records.

## 🛠️ Technology Stack
* **Backend:** C#, classic ASP.NET Web Application (.NET Framework)
* **Database:** Microsoft SQL Server (Relational Database Management System)
* **IDE:** Visual Studio 2022

## 🚀 Installation & How to Run

Because this is a classic ASP.NET Framework project, it requires **Visual Studio 2022** and an instance of **Microsoft SQL Server** to run locally. It utilizes IIS Express for the local development server.

### Prerequisites
* [Visual Studio 2022](https://visualstudio.microsoft.com/) (with the "ASP.NET and web development" workload installed).
* Microsoft SQL Server (or SQL Server Express) and SQL Server Management Studio (SSMS).
